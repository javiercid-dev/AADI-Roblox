# 🔐 Advanced Anomaly Development Institute (AADI)

**AADI** es una base SCP-style SUPERREALISTA para Roblox con 500 anomalías, sistema de dinero, tarjetas, y progresión desbloqueando niveles.

## 📋 Estructura de la Base

```
SCP-AADI-000 → Entrada (Recepción)
    ↓
SCP-AADI-100 → Personal Civil & Oficinas
    ↓
SCP-AADI-200 → Zonas de Armamiento
    ↓
SCP-AADI-300 → Laboratorios Científicos
    ↓
SCP-AADI-400 → Celdas de Contención
    ↓
SCP-AADI-500+ → ABISMO (anomalías más peligrosas)
```

## 🎮 Sistema de Progresión

- **Nivel 0**: Acceso a AADI-1 a AADI-99
- **Nivel 1**: Desbloquea AADI-100 a AADI-199 (estudiando)
- **Nivel 2**: AADI-200 a AADI-299
- **Nivel 3**: AADI-300 a AADI-399
- **Nivel 4**: AADI-400 a AADI-499
- **Nivel 5**: AADI-500 (ABISMO - lo más peligroso)

## 💰 Moneda: A-Credits

Ganas A-Credits por:
- ✅ Estudiar anomalías
- ✅ Completar misiones
- ✅ Trabajar como empleado (médico, soldado, científico)

## 🎯 Características

✅ **500 Anomalías AADI** catalogadas
✅ **Sistema de Tarjetas de Acceso** (escaneo en puertas)
✅ **Puertas Inteligentes** (diferentes niveles de seguridad)
✅ **Tienda de Armas** (rifles, pistolas, kits médicos)
✅ **Empleados con Roles** (médicos, soldados, científicos)
✅ **NPCs con Diálogos SCP-style**
✅ **Sistema de Desbloqueo Progresivo**

## 📁 Archivos Lua

- `AnomaliesDatabase.lua` - 500 anomalías
- `AccessCardSystem.lua` - Tarjetas inteligentes
- `DoorSystem.lua` - Puertas de seguridad
- `WeaponShop.lua` - Tienda con A-Credits
- `EmployeeSystem.lua` - Roles y empleados
- `UnlockSystem.lua` - Desbloqueo progresivo
- `NPCDialogues.lua` - Diálogos SCP-style
- `main.lua` - Script principal

## 🚀 Instalación

1. Abre Roblox Studio
2. Copia todos los archivos `.lua` en **ServerScriptService**
3. Ejecuta `main.lua` primero
4. ¡Disfruta! 🔐
